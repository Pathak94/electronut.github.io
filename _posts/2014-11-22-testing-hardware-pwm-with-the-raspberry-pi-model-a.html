The new $20 Raspberry Pi Model A+ has the same pin-out as the Model B+, which means it has two hardware PWM pins exposed.

These pins are GPIO18 and GPIO13 (BCM pin numbering). This is great news, and I wanted to test it out. So I used Python and wiringPi to generate PWM on these pins, and hooked them up to my oscilloscope. As you can see from the above photo, PWM0 (red) is at 25% duty cycle and PWM1 (yellow) is at 75% duty cycle.
