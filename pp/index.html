Installing Source Code for the Book's Projects

You can download source code for the book's project from the link below:

https://github.com/electronut/pp

Use the “Download ZIP” option the the above link to retrieve the code. Once you get the code you need to add the common folder to your PYTHONPATH environment variable so that modules that use these common python files can find them. On Windows, you can do this by creating a PYTHONPATH environment variable, or adding to one if it already exists. On OS X, you can add the following line to your .profile file (Create one if needed.) in your home directory.

export PYTHONPATH=path_to_common_folder:$PYTHONPATH

Linux users can do something similar to OS X, in their .bashrc or .bash_profile, as appropriate.

Installation Notes for Windows

First, you need to install Python.

Installing Python

You can download and install Python from the website below.

https://www.python.org/download

Installing GLFW

For the OpenGL based 3D graphics projects in this book, you need the GLFW library. You can install it from here.

http://www.glfw.org/download.html

On Windows, after you install GLFW, set a GLFW_LIBRARY environment variable to the full path of the installed glfw3.dll. It will look something like “C:\glfw-3.0.4.bin.WIN32\lib-msvc120\glfw3.dll”. This is needed so that our python binding for GLFW can find this library. To use GLFW with python, we use a module called pyglfw, which consists of a single python file glfw.py. You don't need to install it, as we are distributing it with the source code for the book, and it will be found in the common directory. But just in case you need to install a more recent version, here is the source:

https://github.com/rougier/pyglfw

You also need to ensure that your graphics card drivers on your computer are installed. This is good thing in general, since many software programs (especially) games make use of the GPU.

Installation Notes for OS X

Here are the recommended steps for installation on Apple OS X.

Install Xcode and MacPorts

The first step is to install Xcode. You can either get it through the App Store application, or if you are running an older version of the OS, you can get a compatible version of Xcode from the Apple developer website. Once you install Xcode, you need to ensure that you also have the command line tools installed. The next step is to install MacPorts. Please read the MacPorts guide to help you with this process which has detailed instructions for doing both.

Install Modules

Once you have MacPorts installed, you can install the required modules for the book using the 'port' command in a Terminal. MacPorts installs its own version of Python, and it's simplest to just use that one for your projects. (OS X also comes built in with Python, but installations on top of it are fraught with problems and it's best left alone.)

You can use the following command in a Terminal to check the versions of Python.

$ port select --list python

In case you have multiple Python installations, you can make a particular version of Python active for MacPorts using the following command. (Here it's assumed you want to make python27 as the active one for installations.)

$port select --set python python27

Here is how you can install the required modules. Run these commands one by one in a Terminal.


sudo port install py27-numpy
sudo port install py27-Pillow
sudo port install py27-matplotlib
sudo port install py27-opengl
sudo port install glfw
sudo port install py27-scipy
sudo port install py27-pyaudio
sudo port install py27-serial
sudo port install py27-game

MacPorts usually installs its Python in /opt/local/. You can ensure you get the right version of Python in the Terminal by setting the PATH environment variable in your .profile. Here is how I have it set up.


PATH=/opt/local/Library/Frameworks/Python.framework/Versions/2.7/bin:$PATH
export PATH

Installation Notes for Linux

Linux usually comes built-in with all the development tools needed to build the required packages. On most Linux distributions, you should be able to uses pip to get the required packages for the book. The common way to install a package using pip is as follows:


sudo pip install matplotlib

Once you install pip using the above link, you will also be able to build any of the required packages using the source by running the following command in the source directory.


sudo python setup.py install
